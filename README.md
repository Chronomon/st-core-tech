# What is this
《时空大数据系统核心技术》的配套代码，工程里每个模块和书中的章节相对应。这个代码库的目的是为了帮助读者更快的应用这些算法，因此我们没有过多地深入到算法的细节中，而是着重呈现它们最常用的使用方法。文章内提到的算法都提供了包含Main方法的可执行类，用户可以查看执行类内包含的java工具包，以及使用方法。
